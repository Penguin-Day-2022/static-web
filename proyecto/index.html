Etapas del Proyecto

Página Web

Software

Creación de una página web a través de servidor local, mediante el uso de las siguientes herramientas:

  • Framework en Flask escrito en Python.

  • HTML5 como lenguaje.

  • Tailwind CSS para front-end.

  • GitHub para alojar el proyecto open source.
